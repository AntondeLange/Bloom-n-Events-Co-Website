# Quick Start Guide

Get started with the refactoring in 5 minutes.

## 1. Install Dependencies

```bash
npm install
```

## 2. Optimize Images (Optional - First Time)

```bash
npm run optimize-images
```

This creates optimized WebP/AVIF versions in `assets/images/optimized/`

## 3. Build for Production

```bash
npm run build
```

Outputs optimized files to `dist/` directory.

## 4. Preview Build

```bash
npm run preview
```

Opens preview at `http://localhost:4173`

## 5. Development Mode

```bash
npm run dev
```

Starts dev server at `http://localhost:3000`
